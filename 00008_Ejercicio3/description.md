Veamos  el siguiente código:

``` python
palabra_1 = 'python'

palabra_2 = 'una frase'

if len(palabra_1) > len(palabra_2):
    print(palabra_1)
    
else len(palabra_1) < len(palabra_2):
    print(palabra_2)

```
Comentario: el llamado a len devuelve el largo de la palabra (los espacios también se cuentan)
